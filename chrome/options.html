Help

TabCloud lets you save open windows and restore them later, to do this you use the TabCloud icon on the menu. The floppy disk icon allows you to save open windows, the plus icon on saved windows restores them, you can also use the minus icon to delete saved windows. There are a number of advanced features listed below which allow you to better organise your tabs.

Tips:

  • Click on a window's name to rename it.
  • Middle click on an icon to open just that tab.
  • Drag an icon to re-arrange tabs.
  • Hover on an icon to view the title for that tab.
  • Click on an icon to jump to that tab.
  • Drag a tab to the trashcan icon to remove it.
  • Dragging saved windows lets you reorder them.
